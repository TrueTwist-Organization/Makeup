# Glowelle | Modern 3D Premium Cosmetic Website

I have created a high-end, premium cosmetic website using **React.js (Vite)**, **Framer Motion**, and **React Parallax Tilt**. The design features a modern glassmorphism aesthetic with smooth animations and interactive 3D elements.

## ✨ Key Features

### 1. 🌈 Two Large Interactive Categories
-   **Makeup**: Vibrant rosy gradients with high-energy 3D tilt cards.
-   **Skincare & Haircare**: Clean, minimal sage-green palette with soft transitions.

### 2. 🧊 3D Hover & Glassmorphism
-   **Tilt Effects**: Every product card and category card tilts and rotates on mouse move for a truly interactive 3D feel.
-   **Glassmorphic Navbar & Footer**: Deep blur effects and soft white-transparent backgrounds for a premium look.

### 3. 🛡️ Premium Components
-   **Navbar**: Responsive menu with glass effect and integrated cart count.
-   **Product Cards**: High-quality product images, automatic star ratings, and price tags with a floating "Add to Cart" button.
-   **Smooth Scrolling**: Built-in "Scroll to Top" for seamless navigation.

### 4. 🗃️ Dynamic Routing
-   **React Router**: Effortless navigation between the home page and category-specific galleries.
-   **AnimatePresence**: Future-proofed for entry/exit animations between pages.

## 🚀 How to Run Locally

If you are not already running the development server:

1.  Open your terminal.
2.  Navigate to the project folder (`f:\makeup`).
3.  Run the following commands:
    ```bash
    npm install
    npm run dev
    ```
4.  Open [http://localhost:5173](http://localhost:5173) in your browser.
